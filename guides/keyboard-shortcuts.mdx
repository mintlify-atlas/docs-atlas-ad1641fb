Master En Croissant's keyboard shortcuts to analyze games faster and more efficiently. All shortcuts can be customized in Settings.

## File Operations

| Action | Mac | Windows/Linux | Description |
|--------|-----|---------------|-------------|
| New tab | `Cmd+T` | `Ctrl+T` | Create a new analysis tab |
| Close tab | `Cmd+W` | `Ctrl+W` | Close the current tab |
| Open file | `Cmd+O` | `Ctrl+O` | Open a PGN file |
| Save file | `Cmd+S` | `Ctrl+S` | Save current game/analysis to PGN |
| Settings | `Cmd+,` | `Ctrl+,` | Open application settings |
| Reload app | `Cmd+R` | `Ctrl+R` | Reload the application |

<Tip>
  Press `Cmd+T` / `Ctrl+T` quickly to start analyzing a new position or create a new repertoire file.
</Tip>

## Tab Navigation

| Action | Shortcut | Description |
|--------|----------|-------------|
| Cycle tabs forward | `Ctrl+Tab` | Switch to the next tab |
| Cycle tabs backward | `Ctrl+Shift+Tab` | Switch to the previous tab |

<Note>
  Tab cycling wraps around - pressing next on the last tab goes to the first tab.
</Note>

## Move Navigation

Navigate through games and variations efficiently:

### Basic Navigation

| Action | Shortcut | Description |
|--------|----------|-------------|
| Next move | `‚Üí` | Move forward one move |
| Previous move | `‚Üê` | Move backward one move |
| Go to game start | `Shift+‚Üë` | Jump to the starting position |
| Go to game end | `Shift+‚Üì` | Jump to the final position |

### Variation Navigation

| Action | Shortcut | Description |
|--------|----------|-------------|
| Go to branch start | `‚Üë` | Jump to the start of the current variation |
| Go to branch end | `‚Üì` | Jump to the end of the current variation |
| Next variation | `C` | Switch to the next sibling variation |
| Previous variation | `X` | Switch to the previous sibling variation |
| Next branching point | `Shift+‚Üí` | Jump to the next move with variations |
| Previous branching point | `Shift+‚Üê` | Jump to the previous move with variations |

<Accordion title="Understanding Variation Navigation">
  **Branch navigation** (`‚Üë` and `‚Üì`):
  - Moves within the current variation line
  - Useful for quickly reviewing a single variation
  
  **Variation switching** (`C` and `X`):
  - Switches between alternative moves at the current position
  - Stays at the same position, changes which line you're viewing
  
  **Branching point jumps** (`Shift+‚Üê` and `Shift+‚Üí`):
  - Skips to positions where the game has multiple continuations
  - Efficient for finding critical decision points
</Accordion>

### Editing

| Action | Shortcut | Description |
|--------|----------|-------------|
| Delete move | `Delete` / `Backspace` | Delete the current move and all following moves in the variation |

<Warning>
  Deleting a move is permanent and cannot be undone. Save your file frequently.
</Warning>

## Move Annotations

Add standard chess annotations instantly:

| Annotation | Shortcut | Symbol | Meaning |
|------------|----------|--------|----------|
| Brilliant move | `1` | ‚Äº | Exceptional, hard-to-find move |
| Good move | `2` | ! | Strong move |
| Interesting move | `3` | !? | Enterprising, unclear consequences |

## Repertoire Practice

Shortcuts active during practice mode:

| Action | Shortcut | Description |
|--------|----------|-------------|
| Rate "Again" | `1` | Mark position as completely forgotten |
| Rate "Hard" | `2` | Mark position as difficult to recall |
| Rate "Good" | `3` | Mark position as remembered with effort |
| Rate "Easy" | `4` | Mark position as instantly recalled |
| Skip position | `Space` | Move to next position after incorrect answer |
| Toggle blur | `Ctrl+B` | Show/hide piece visibility during practice |

<Accordion title="Rating Shortcuts in Practice">
  After answering correctly, use number keys `1-4` to rate difficulty:
  
  - **`1` (Again)**: You forgot completely ‚Üí Review soon (minutes/hours)
  - **`2` (Hard)**: You struggled ‚Üí Review in days
  - **`3` (Good)**: Recalled with effort ‚Üí Review in weeks
  - **`4` (Easy)**: Instant recall ‚Üí Review in months
  
  Accurate ratings optimize your review schedule for long-term retention.
</Accordion>

## Customizing Shortcuts

All keyboard shortcuts can be customized:

<Steps>
  <Step title="Open Settings">
    Press `Cmd+,` (Mac) or `Ctrl+,` (Windows/Linux).
  </Step>
  
  <Step title="Navigate to Keybinds">
    Find the "Keybinds" or "Keyboard Shortcuts" section.
  </Step>
  
  <Step title="Click on Shortcut">
    Click any shortcut to record a new key combination.
  </Step>
  
  <Step title="Press New Keys">
    Press your desired key combination. It will be validated for conflicts.
  </Step>
  
  <Step title="Save Changes">
    Changes are saved automatically.
  </Step>
</Steps>

<Warning>
  Avoid conflicts with system shortcuts (like `Cmd+Q` for quit) or En Croissant may not receive the key press.
</Warning>

## Shortcut Tips and Tricks

  <Card title="Chain Commands" icon="link">

  </Card>
  
  <Card title="Panel Switching" icon="window">
    Use letter keys (`P`, `A`, `B`, `D`, `I`) to quickly switch between analysis contexts without mouse clicks.
  </Card>
  
  <Card title="Arrow Navigation" icon="arrows">
    Master arrow keys for move navigation. With Shift, they become game/branch jumpers. This is the fastest way to navigate.
  </Card>
  
  <Card title="Annotation Workflow" icon="pen">
    After analyzing with `A`, switch to `D` and use `1-6` to annotate moves, then `‚Üê/‚Üí` to move through the game.
  </Card>
</CardGroup>

## Platform-Specific Notes

### macOS
- `Cmd` is used for most shortcuts (instead of `Ctrl`)
- Some shortcuts may conflict with system features (Mission Control, Spotlight)
- Customize system shortcuts in System Preferences if needed

### Windows/Linux
- `Ctrl` is the primary modifier key
- `Alt` may be used for some secondary shortcuts
- Window managers may intercept some key combinations
